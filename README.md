# Paper-Portal-Site
This repo is created to experiment claude design with an imaginary second hand books marketplace's site

## hosted link
[Paper Portal](https://adorable-meringue-cd0a77.netlify.app/)

## How this created using claude design 

### Chatgpt Part 
1. Got the idea of second hand books market place
2. Made a story emphasing books' value and sustanability aspect of reusing books 
3. Added the context and UI flow to chatgpt and asked to generate scenes for the web site as story telling flow and decide a theme 
4. Then asked to generate images for each scene
5. Then ask to provide a summary prompt to decide this to other AI application which claude design 

### Claude Design 
1. Upload the resources and provide the chatgpt generated prompt and add some UI animation instructions like parallex effect, scrolling telling flow 
2. Then claude generated questions to get more idea before design. Answer to those questions. You can get tweeks to compare. 
3. Then it will generate the design 
4. Comment, revamp and ask for the modifications until it gives exact output you want. You can refer the section and comment. For this site, needed 4,5 revamps. 
5. Then ask a way to deploy this. and then it gives the single page bundled file (index.html). Since this is a static single page application I deployed using netflix drop. Additionally we can use github pages, vercel, s3 etc.
